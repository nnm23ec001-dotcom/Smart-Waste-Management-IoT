**High-level wiring**

- HC-SR04 TRIG -> Pico GP2
- HC-SR04 ECHO -> Pico GP3 (use a 1k/2k divider for 5V->3.3V if needed)
- DHT22 data -> Pico GP4 (with pull-up resistor)
- MQ2 analog -> Pico ADC0 (GP26)
- SSD1306 SCL -> GP1, SDA -> GP0 (I2C)
- SG90 Servo -> GP5 (PWM) ; servo Vcc to 5V, GND common
- Buzzer -> GP6
- LED_GREEN -> GP7
- LED_RED -> GP8
